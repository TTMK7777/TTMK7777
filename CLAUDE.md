# TTMK7777 - Claude Code Spec-Driven Workflow

## Overview
This is the GitHub profile README repository for TTMK7777. It displays a personal profile page on the GitHub user profile (github.com/TTMK7777).

## Project Structure

| Path | Description |
|------|-------------|
| `README.md` | GitHub profile README (auto-displayed on profile page) |
| `specs/` | Feature specifications for profile enhancements |
| `spec.md` | Project specification |
| `plan.md` | Development plan and phase tracking |
| `todo.md` | Task tracking |
| `knowledge.md` | Technical decisions and learnings |
| `memory.md` | Failures, lessons learned, rejected approaches |

## Content Conventions

- **Format**: Markdown (GitHub Flavored Markdown)
- **Badges**: shields.io format
- **Stats**: github-readme-stats.vercel.app
- **Tone**: Professional, concise
- **Language**: English (primary)

## Workflow

1. Check `spec.md` for profile content requirements
2. Check `plan.md` for current priorities
3. Check `todo.md` for task status
4. Edit README.md following content conventions
5. Update `todo.md` and `knowledge.md` as needed
