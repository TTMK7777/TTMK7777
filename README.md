## How I Work

- **Strategy first, code second.** I write specs and ADRs before lines of code, then automate the implementation.
- **Multi-agent by default.** Single-LLM workflows leave value on the table for non-trivial tasks. I architect for 2–4 collaborating agents with explicit role boundaries.
- **Security as a CI gate.** AI-generated code needs an automated guardrail — that's why I built `code-validator`.
- **Open source what scales.** Internal tools that prove valuable get pulled out and released MIT.
